# NeoTus
A Discord bot made in Python with Discord.py
To play Motus.

## Prerequisites

- Python 3 (I am using 3.9.1)
- with the Discord.py module (I am using 1.7.3)
and
- aiohttp
- async-timeout
- attrs
- certifi
- chardet
- charset-normalizer
- idna
- multidict
- python-dotenv
- requests
- typing_extensions
- Unidecode
- urllib3
- yarl
- bs4
- platform
